Projects

  • Auto recipe tagging

    In this self-driven project I predict upto 10 of the most popular tags for any given recipe. The NB-SVM model, created using sciki-learn library, obtained an ROC-AUC score of 0.788 in this multilabel problem. Dataset was prepared by scraping the site epicurious.com. Code for this project is available in this github repository.

  • Bayesian Optimization using Gaussian Processes

    As part of my project for CSE569, I implemented Bayesian Optimization using Gaussian Processes, for tuning the hyperparameters of a Convolutional Neural Network. Our models were tested on the Street View House Numbers dataset and obtained an accuracy of 82.85% on the test set.

  • Semantic Parsing the ARC dataset

    We performed semantic parsing of the ARC dataset, which is a dataset containing grade school level MCQ with 4 answers, as part of CSE576. We also built a BERT based model with an IR system, for the question answering task. The model gave an accuracy of 0.317 on the challenge dataset.

  • Authentication using EEG data

    For CSE535, we built a mobile app that would authenticate users using their EEG data. The app will run an ML model on, the cloud, the mobile, or on a fog server, whichever has been identified to return the result the fastest. We used Tensorflow to create an LSTM model and a CNN model to identify the user from their EEG data.

  • Topic Modelling

    Extracted topics from the Twenty newsgroups and Enron datasets using Latent Dirichlet Allocation and Hierarchical Dirichlet Process. I also benchmarked the performance of gensim and mallet libraries in extracting relevant topics. Code for this self-driven project is available in this github repository.
